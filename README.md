# V3 足球赔率预测系统

基于历史赔率数据的足球比赛预测系统，使用V3模型进行完整分析。

## 功能特点

- ✅ **V3模型完整分析**：7步骤完整分析流程
- ✅ **双机构赔率对比**：支持Bet365和William Hill双机构数据
- ✅ **本地网页界面**：无需联网，本地运行
- ✅ **历史数据支持**：16000+场比赛历史数据
- ✅ **详细分析报告**：包含胜平负、大小球、净胜球分布等

## 技术栈

- **后端**：Python 3 + Flask
- **前端**：HTML + CSS + JavaScript
- **数据库**：SQLite

## 使用方法

### 输入赔率数据

在网页界面中输入以下数据：

**Bet365赔率：**
- 初盘：主胜 / 平赔 / 客胜
- 临场：主胜 / 平赔 / 客胜

**William Hill赔率：**
- 初盘：主胜 / 平赔 / 客胜
- 临场：主胜 / 平赔 / 客胜

**大小球盘口：**
- 初盘：盘口 / 赔率
- 临场：盘口 / 赔率

### 查看预测结果

点击"开始预测"按钮后，系统将：

1. 进行7步骤完整分析
2. 生成详细预测报告
3. 提供胜平负预测和置信度
4. 提供大小球预测
5. 推荐最可能比分
6. 风险提示和关键信号

## V3模型分析流程

### 步骤1：初盘+临场完全匹配（最重要）
- 6个层级完全匹配分析
- 双机构初盘+临场完全一致
- 双机构临场完全一致
- 双机构主胜临场一致
- 单机构临场完全一致
- 双机构主胜临场接近

### 步骤2：相似度分析（核心筛选）
- 双机构综合相似度分层
- 3个相似度区间分析

### 步骤3：临场精确匹配（辅助）
- 7个层级临场赔率匹配

### 步骤4：杠杆率分析（辅助）
- B365杠杆率分层（6层级）
- WH杠杆率分层（6层级）

### 步骤5：相似度分层分析（辅助）
- 综合相似度分层（5层级）
- 趋势分析

### 步骤6：爆冷规律分析（风险提示）
- 确定临场赔率最小方
- 筛选基础样本（±0.03）
- 平局组合TOP 10
- 客胜组合TOP 10
- 当前赔率风险评估

### 步骤7：综合预测输出
- 主要依据汇总
- 辅助依据汇总
- 最终预测和置信度
- 推荐比分
- 风险提示

## 数据库说明

系统使用SQLite数据库，包含16000+场历史比赛数据：
- 5大联赛数据（英超、西甲、德甲、意甲、法甲）
- Bet365和William Hill双机构赔率
- 初盘和临场赔率
- 比赛结果和比分数据
