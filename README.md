# V3足球赔率预测系统 - 优化版

基于历史赔率数据的足球比赛预测系统，包含完整的V3模型分析流程。

## 🔥 最新优化（2026-06-22）

### 新增功能

1. **杠杆率预警系统** ⭐
   - 杠杆率≥3.0时自动预警
   - 基于三场横向对比分析发现的关键规律
   - 自动调整预测置信度

2. **双机构一致性评级** ⭐
   - 🔥🔥🔥 完全一致（差异<0.01）
   - 🔥🔥 接近（差异≤0.05）
   - ⚠️ 分歧较大（差异>0.05）

3. **风险评估系统**
   - 综合杠杆率、一致性、爆冷风险
   - 自动生成预警信息
   - 可视化风险等级

### 关键发现

通过三场比赛横向对比分析发现：

| 杠杆率区间 | 样本量 | 主胜率 | 平局率 | 风险级别 |
|-----------|--------|--------|--------|---------|
| **≥3.0** | 1646场 | 74.8% | **15.4%** ⚠️ | 高风险 |
| **<3.0** | 265场 | 69.8% | 18.1% | 相对安全 |

**核心规律**：杠杆率≥3.0时，平局风险显著上升！

## 快速开始

### 1. 安装依赖

```bash
pip install flask
```

### 2. 准备数据库

将赔率数据库放到 `data/football_odds_complete.db`

### 3. 启动服务

```bash
cd v3-predictor
python app.py
```

访问 http://localhost:5000

### 4. 使用API

```python
import requests

data = {
    'b365_home_open': 1.67,
    'b365_draw_open': 3.70,
    'b365_away_open': 5.00,
    'b365_home_close': 1.44,
    'b365_draw_close': 4.33,
    'b365_away_close': 7.50,
    'wh_home_open': 1.55,
    'wh_draw_open': 3.80,
    'wh_away_open': 5.50,
    'wh_home_close': 1.44,
    'wh_draw_close': 4.20,
    'wh_away_close': 7.00,
    'ou_open': 2.5,
    'ou_close': 2.5
}

response = requests.post('http://localhost:5000/predict', json=data)
result = response.json()
```

## V3模型完整步骤

### 步骤1：初盘+临场完全匹配（最重要）

- 层级1.3：双机构主胜临场一致
- 层级1.6：双机构主胜临场接近（±0.05内）

### 步骤2：相似度分析（核心筛选）

完整5分组：
- 分组1：相似度90%-100%
- 分组2：相似度80%-90%
- 分组3：相似度70%-80%
- 分组4：相似度60%-70%
- 分组5：相似度50%-60%

### 步骤3：临场精确匹配（辅助）

完整7层级，每个层级都包含主胜净胜球分布：
- 层级3.1：双机构主胜临场完全一致
- 层级3.7：双机构主胜临场接近（±0.05内）

### 步骤4：杠杆率分析（辅助）

B365和WH分别分析（各6层级）：
- 以目标杠杆率为中心，±0.10为单位分层
- 杠杆率≥3.0自动预警

### 步骤6：爆冷规律分析（风险提示）

优化版逻辑：
1. 确定临场赔率最小的一方
2. 筛选最小赔率±0.03范围内的比赛
3. 分析平局组合（TOP 10）
4. 分析客胜/主胜组合（TOP 10）
5. 检查当前赔率组合的爆冷概率

### 步骤7：综合预测输出

- 主要依据汇总
- 最终预测（含置信度调整）
- 大小球分析
- 推荐比分
- 风险提示
