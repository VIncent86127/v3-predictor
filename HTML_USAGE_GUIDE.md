## 🔥 关键功能说明

### 1. 杠杆率预警系统

**杠杆率计算：**
```
杠杆率 = 平赔 ÷ 主胜
```

**预警规则：**
- 杠杆率 ≥ 3.0：高风险（平局预警）
- 杠杆率 2.5-3.0：中风险
- 杠杆率 < 2.5：低风险

**自动调整：**
- 杠杆率≥3.0时，置信度自动降低3%

### 2. 双机构一致性评级

**评级标准：**
- 🔥🔥🔥 完全一致：差异 < 0.01
- 🔥🔥 接近：差异 ≤ 0.05
- ⚠️ 分歧较大：差异 > 0.05

**自动调整：**
- 完全一致：置信度提升2%
- 分歧较大：置信度降低2%

### 3. 风险评估

**综合评估：**
- 杠杆率
- 双机构一致性
- 爆冷概率
- 当前赔率组合风险

**风险级别：**
- HIGH：高风险（红色）
- MEDIUM：中风险（黄色）
- LOW：低风险（绿色）

## ⚠️ 注意事项

### 1. 必须先启动服务器

HTML文件需要连接后端API才能工作，使用前必须先启动：

```bash
cd /home/ubuntu/.openclaw/workspace/v3-predictor
./start.sh
```

### 2. 浏览器兼容性

推荐使用：
- Chrome / Edge / Firefox（最新版本）
- Safari（macOS）

### 3. 数据库要求

确保数据库文件存在：
```
/home/ubuntu/.openclaw/workspace/football_odds_complete.db
```

### 4. 端口占用

如果5001端口被占用，修改 `app.py`：
```python
app.run(debug=True, host='0.0.0.0', port=5002)
```

然后修改HTML文件中的API地址：
```javascript
const API_URL = 'http://localhost:5002';
```

## 🛠️ 常见问题

### Q1: 显示"API未连接"

**解决方法：**
```bash
# 1. 检查服务是否启动
ps aux | grep app.py

# 2. 重新启动服务
cd /home/ubuntu/.openclaw/workspace/v3-predictor
./start.sh

# 3. 检查端口
netstat -tlnp | grep 5001
```

### Q2: 预测失败

**可能原因：**
1. 数据库文件不存在
2. 赔率数据格式错误
3. 服务器未启动

**解决方法：**
- 检查数据库文件路径
- 确保赔率为有效数字
- 重启服务器

### Q3: 页面加载慢

**优化方法：**
- 使用本地CDN资源
- 关闭不必要的浏览器扩展
- 使用Chrome浏览器
